# BOOM-Explorer
BOOM Explorer: RISC-V BOOM Microarchitecture Design Space Exploration


## Usage
- Get help
```bash
$ python main.py -h
```
- Run BOOM-Explorer
```bash
$ python main.py -c configs/boom-explorer.yml
```
